# Components Of Vue

## 介绍
是基于 Vue 的 UI 组件库，主要用于大家的积累和展示。

## 特性

- 🥳开箱即用的高质量 Vue 组件。
- ⚙️100+ 高质量组件
- 👹有丰富的单元测试，每个组件都经过严格的测试
- 🌵自说明的代码，即使没有注释，你也能看懂

## 安装
### npm 安装
推荐使用 npm 的方式安装，它能更好地和 [webpack](https://webpack.js.org/) 打包工具配合使用。
```
npm i gulu-front-components -S
// or yarn
yarn add gulu-front-components
```
